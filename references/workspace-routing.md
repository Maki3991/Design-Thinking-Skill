# Workspace Routing

This skill is designed to be self-contained.

## Default Source Order

1. Read packaged references in this skill first.
2. Always start from `router.md`.
3. Read `DT1.md` only for intro, life-design, or Lesson 1 concept questions.
4. If the active workspace also contains related DT files, use them only as supplements.
